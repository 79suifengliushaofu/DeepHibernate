# DeepHibernate 更新日志

## v2.0 (2026-07-15) — 重新编译

### v2.0 经典 UI + 新特性

- **保留 v2.0 四个选项框**：
  - 关闭 USB 选择性暂停（默认勾选）
  - 禁用设备唤醒权限（默认勾选）
  - 关闭网卡 Wake-on-LAN（默认不勾选）
  - 禁用计划任务唤醒（默认勾选）
- **可自定义目标温度**：滑块 + 数字输入，20~80°C，默认 35°C
- **实时双温度显示**：CPU（WMI MSAcpi_ThermalZoneTemperature）+ GPU（nvidia-smi），带颜色指示
- **主动散热电源策略**：进入降温前自动切换高性能电源方案加速散热
- **零依赖**：回归系统原生 API，20KB 单文件

### 技术变更

- 源码从 v1.0 重新编写：WinForms .NET Framework 4.x / C# 5
- 新增 `System.Management.dll` 引用（WMI 温度查询）
- 使用 `BackgroundWorker` 实现非阻塞降温循环
- TrackBar + NumericUpDown 双向同步

## v4.0 (2026-07-15)

### 极简化重构

- **极简界面**：仅保留目标温度设置（20~80C，默认 35C）+ 实时 CPU/GPU 温度显示
- **去除冗余**：移除所有选项框（键鼠唤醒、WOL、模式选择等），参数全部内置为合理默认值
- **实时双温度**：CPU 通过 WMI MSAcpi_ThermalZoneTemperature，GPU 通过 nvidia-smi（不可用时回退 rocm-smi）
- **内置默认参数**：S0 现代待机、检测间隔 5 秒、超时 300 秒、高性能电源方案
- **暗色极简 UI**：深色主题，视觉聚焦核心信息
- **流线型操作**：设置目标温度 → 点击按钮 → 自动散热降温 → 达标后进入睡眠
- **零依赖**：去掉 LibreHardwareMonitorLib.dll，回归系统原生 API，16.5KB 单文件

## v3.0 (2026-07-15)

### 重大更新：从脚本到独立应用

- **全新 GUI 配置面板**：替代原有的手动编辑 JSON 配置文件方式，提供直观的图形化操作界面
- **可自定义参数**：目标温度、等待时间、检测间隔均可通过 GUI 自由调节
- **双模式睡眠支持**：
  - S0 现代待机（Modern Standby）：适合需要快速唤醒的场景
  - S4 休眠（Hibernate）：适合长时间离开，更节能
- **主动散热电源策略**：进入休眠前自动切换高性能电源方案，加速散热
- **实时温度显示**：GUI 面板实时刷新 CPU/GPU 温度，一目了然
- **JSON 配置持久化**：所有设置保存至 `DeepHibernate_Config.json`，重启后自动加载
- **应用图标**：嵌入专属 DeepHibernate 图标（16x16 ~ 256x256 多尺寸）

### 技术变更

- 从 PowerShell 脚本 (.ps1) 迁移为独立可执行文件 (.exe)，无需依赖 PowerShell 环境
- 集成 LibreHardwareMonitorLib 实现硬件温度监控
- 支持 Windows 10/11 全版本
